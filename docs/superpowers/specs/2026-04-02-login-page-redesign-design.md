# Cocoon Break 登录页重新设计

## 1. 概念与愿景

Cocoon Break 是一个 AI SuperAgent 平台，"茧"象征用户当前的困境/局限，"破茧"代表 AI 帮助用户突破、释放生产力。登录页作为用户的第一印象，需要传达温暖、希望、新生的感觉，让用户感受到"进入后将被赋能"。

**设计主题**：晨曦破晓 — 温暖的橙黄渐变天空下，蝴蝶从茧中破壳而出，飞向新生。

## 2. 设计语言

### 美学方向
自然生命风 + 晨曦暖色调。参考破茧成蝶的生命周期，传达希望与蜕变。

### 色彩系统
| 角色 | 色值 | 用途 |
|------|------|------|
| 背景深橙 | #FF8C42 | 背景底部 |
| 背景金黄 | #FFD700 | 背景中层 |
| 背景淡粉 | #FFB6C1 | 背景顶部 |
| 光晕中心 | #FFF8DC | 径向光晕核心 |
| 文字主色 | #FFFFFF | 标题、输入框文字 |
| 文字次色 | rgba(255,255,255,0.7) | placeholder、次要文字 |
| 卡片背景 | rgba(255,255,255,0.1) | 毛玻璃登录卡片 |
| 卡片边框 | rgba(255,255,255,0.2) | 卡片边框 |
| 按钮主色 | #FF8C42 | 登录按钮背景 |
| 按钮悬停 | #FFA060 | 按钮 hover |

### 字体
- **主标题**: 系统默认无衬线字体，字重 700，渐变填充
- **副标题/正文**: 系统默认无衬线字体，字重 400
- **输入框**: 等宽字体风格暗示安全感

### 空间系统
- 登录卡片最大宽度：420px
- 卡片内边距：40px（桌面）/ 24px（移动）
- 元素间距：16px / 24px 节奏
- 圆角：24px（大卡片）/ 12px（输入框/按钮）

### 动效哲学
- **破茧动画**：缓慢、庄严、有张力，暗示蜕变的重量
- **飞舞动画**：轻盈、自由、循环，象征解脱后的自由
- **UI 交互**：微妙的 hover/press 反馈，温暖不突兀

## 4. 功能与交互

### 登录表单
- **用户名输入框**：图标 + placeholder "用户名"，回车跳转到密码框
- **密码输入框**：图标 + placeholder "密码"，type=password，显示/隐藏切换按钮
- **登录按钮**：文字 "进入工作区"，点击后显示 loading 状态
- **表单验证**：空字段提交时抖动 + 红色边框提示

### 蝴蝶动画序列
1. **Phase 1 — 破茧**（持续 3 秒，页面加载后播放一次）
   - 茧壳从底部中央出现
   - 茧壳缓慢裂开
   - 蝴蝶从中缓缓爬出，翅膀折叠
   - 翅膀逐渐展开至完全状态

2. **Phase 2 — 飞舞**（破茧完成后循环）
   - 蝴蝶向上飘动，伴随轻微左右摇摆
   - 翅膀以较低频率扇动
   - 到达画面中上部后，渐隐并重新从底部飘出
   - 飘动路径略带曲线，非直线

### 错误处理
- **登录失败**：卡片轻微晃动 + 红色提示文字 "用户名或密码错误"
- **网络错误**：提示文字 "网络连接失败，请重试"

## 5. 组件清单

### LoginCard
- 毛玻璃背景：`backdrop-blur-xl bg-white/10`
- 边框：`border border-white/20`
- 圆角：`rounded-3xl`
- 阴影：`shadow-2xl`
- **States**:
  - Default：正常显示
  - Loading：登录按钮显示 spinner，输入框禁用
  - Error：整体轻微晃动（shake animation）

### InputField
- 背景：`bg-white/10`
- 边框：`border border-white/20`
- 圆角：`rounded-xl`
- 图标：左侧用户/锁图标
- **States**:
  - Default：白色/10 背景
  - Focus：白色/20 背景 + 白色边框
  - Error：红色边框 + 抖动
  - Disabled：opacity 50%

### PrimaryButton
- 背景：`bg-gradient-to-r from-[#FF8C42] to-[#FFB6C1]`
- 圆角：`rounded-xl`
- 文字：白色，字重 600
- **States**:
  - Default：渐变背景
  - Hover：亮度增加 + scale(1.02)
  - Active：scale(0.98)
  - Loading：显示 spinner，文字变为 "登录中..."
  - Disabled：opacity 50%，cursor not-allowed

### ButterflyAnimation
- 实现方式：CSS Keyframes + SVG 或 Lottie
- 破茧阶段：3s ease-out
- 飞舞阶段：8s ease-in-out infinite
- 翅膀扇动：0.3s ease-in-out infinite（与飞舞叠加）

## 6. 技术方案

### 前端框架
- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4

### 实现方式
- 背景：CSS 渐变 + 径向光晕
- 蝴蝶动画：CSS @keyframes + SVG 蝴蝶路径
- 毛玻璃效果：`backdrop-blur-xl`
- 动画性能：使用 `will-change` 优化
